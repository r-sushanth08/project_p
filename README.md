## Smart Newspaper Web-Application

This project focuses on building a **smart newspaper web application** by integrating technologies such as **Python, APIs, HTML, CSS, JavaScript, and backend logic**. 

## Features

- Fetches news using API
- Displays categorized articles
- Responsive UI using CSS
- Backend powered by Python

- The primary objective is to create a functional platform that aggregates and presents news dynamically while serving as a hands-on learning experience. 

## Objective
- Throughout the development process, we aim to strengthen our understanding of **full-stack development** by:

- Applying theoretical concepts in a practical environment
- Iterating on features 
- Improving both the system and our technical skills.
